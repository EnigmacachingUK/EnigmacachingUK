How to set up your Machine for "Enigma One (Training Programme)".

 

Step #1

Open the Enigma Machine and select the appropriate Reflector (UKW)

For this cache Wehrmacht Reflector B is used.

  

Step #2

Now remove the default rotors and place them in the draw with the spare rotors in the base of the machine

 

Step #3

For this cache we are going to use Rotors V, IV and II in that order/sequence  (Walzenlage)

 Select one of the rotors and check its "Ring Setting" (Ringstellung)

For this cache the default setting of "A-01" is used on all 3 rotors.

Install this rotor in its correct position.
Repeat this step until all three rotors are installed.

Ensure that you have installed the
correct rotors in the right order (Walzenlage)

 

Step #4

Close the box and turn the numbers to their correct "Start Position" or "Grundstellung".
For this cache the "Start Position" is- B , B , C

This is represented by 2, 2, 3 on the wheels (HINT: click on one of the the numbers for a pop up alphabet)  

 

You are now set up to break the code for"Enigma One" ....
